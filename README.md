# iFundWatch

iFundWatch 是一个本地单文件基金持仓看板，用来查看场外基金和场内 ETF/LOF 的实时行情、估值和持仓盈亏。

项目不需要后端服务，也不需要构建步骤。直接用浏览器打开 `index.html` 即可使用。

## 功能

- 手动添加持仓或观察基金
- 支持场外基金、场内基金/ETF/LOF、自动识别类型
- 展示当日实时净值估值、涨跌幅、参考净值/昨收、成交额和更新时间
- 根据份额和成本价计算持仓市值、成本、浮动盈亏和收益率
- 支持持仓和观察列表切换
- 支持搜索代码、名称或备注
- 支持编辑、删除、持仓转观察、观察转持仓
- 支持本地 JSON 导入/导出备份
- 数据保存在浏览器 `localStorage`，不会上传到任何服务器

## 使用方法

在浏览器中打开：

```text
/Users/logic/quant/iFundWatch/index.html
```

也可以在 Finder 中双击 `index.html`。

## 添加基金

在左侧表单填写：

- `基金代码`：6 位基金代码，例如 `161725`、`513500`、`159915`
- `类型`：可选 `自动识别`、`场外基金`、`场内基金/ETF/LOF`
- `名称`：可选，行情返回后会自动补全
- `份额`：填写后进入“持仓”，留空则进入“观察”
- `成本价`：可选，填写后用于计算盈亏
- `分组/备注`：可选

保存后页面会自动刷新行情。

## 行情说明

场外基金：

- 使用天天基金公开接口
- `当日实时净值估值` 显示盘中估值
- `参考净值` 显示最近确认的单位净值
- 盘中估值不是官方最终净值，仅用于实时参考

场内基金/ETF/LOF：

- 优先使用东方财富行情接口
- 东方财富不可用时使用腾讯行情接口兜底
- `当日实时净值估值` 显示场内最新交易价
- `参考净值/昨收` 显示昨收价

## 刷新策略

- 交易时段默认每 30 秒刷新一次
- 非交易时段默认每 5 分钟刷新一次
- 可以点击右上角“刷新”按钮手动刷新

## 备份与恢复

点击右上角：

- `导出`：导出当前本地数据为 JSON
- `导入`：从 JSON 备份恢复数据

导入会替换当前浏览器里的本地数据。

## 隐私

持仓、成本价、备注等数据只保存在当前浏览器的 `localStorage` 中。页面不会把你的持仓数据上传到任何服务器。

外部请求仅用于查询基金行情。

## 注意

本项目只用于个人看盘和估算，不构成投资建议。基金估值、场内价格和接口可用性可能受交易时段、数据源延迟或公开接口调整影响。
